1. maxlength : textarea, input 등 텍스트를 입력할 때 글자수 입력제한 기능을 주고 싶을 때 사용
2. visibleRowNum : gridView에서 화면에 포현될 행 수, all로 설정 시 모든 row는 스크롤 없이 표현
3. wheelRows : 마우스 휠을 이용하여 세로 스크롤이 이동 될 때, 한 번에 이동하는 row의 개수
4. wheelStop : 그리드 스크롤이맨 아래로 내려갔을 때 wheel을 한 경우, 브라우저 스크롤을 막을지에 대한 여부
5. readOnly :  그리드 뷰에서 오직 읽는 기능만 제공
6. TabControl => Tabs => Content => WFrame => src : url 링크를 넣으면 탭 클릭시 해당 url 페이지가 랜더링 된다.
