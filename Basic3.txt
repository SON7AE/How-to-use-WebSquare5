// 웹스퀘어5 개발 가이드 - LG Uplus 전사시스템구축 프로젝트 기준

1. 서버관리 이클립스(eclipse) 실행
2. 개발관리 이클립스(eclipse) - 웹스퀘어5 툴 실행
  => SpringToolSuite3와 같은 이름일 수 있다.
3. 위 두 프로그램에서 local server를 실행시킨다.
4. 웹브라우저에 localhost:8090과 같이 해당 로컬서버 주소를 기입하여 화면단이 작동하는지 확인한다.
5. SDM(Service Definition Manager)를 활용하여 각각의 API Data를 활용하여 개발한다.
6. Swagger에 Test해가면서 SDM과 비교하여 개발한다.
